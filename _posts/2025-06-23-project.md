### Redis简介

Redis是一个基于内存的key-value结构数据库

· 基于内存存储，读写性能高

· 适合存储热点数据（热点商品、资讯、新闻）

（对于mysql的补充）

读写性能更高

在文件夹位置处输入cmd即可进入命令行

ctrl+c结束进程

连接redis服务：

```
redis-cli.exe
```

需要在redis-server.exe运行的条件下在另一个cmd窗口运行





### 基于脚手架创建前端工程



```
vue create vue-demo-1
```



```
vue ui
```



node_modules:当前项目依赖的js包

assets:静态资源存放目录

components:公共组件存放目录

App.vue:项目的主组件，页面的入口文件

main.js:整个项目的入口文件

package.json"项目的配置信息、依赖包管理

vue.config.js:vue-cli配置文件

#### 启动前端项目

```
npm run serve
```
